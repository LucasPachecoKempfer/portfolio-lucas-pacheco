# 🚀 Portfólio Pessoal - Lucas Pacheco Kempfer

Este é o repositório do meu portfólio pessoal, um projeto desenvolvido para centralizar minha trajetória, projetos realizados e facilitar o contato profissional. A interface foi construída com foco em modernidade, utilizando efeitos de transparência e animações fluidas.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza o que há de mais essencial no desenvolvimento web front-end:

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: 
    * **Glassmorphism**: Efeitos de desfoque e transparência.
    * **Flexbox & Grid**: Layout responsivo e organizado.
    * **Keyframes**: Animações personalizadas de flutuação.
* **JavaScript (Vanilla)**: Lógica para integração com a API do WhatsApp.

---

## ✨ Funcionalidades

* **Navegação Inteligente**: Menu fixo com *scroll* suave.
* **Fundo Dinâmico**: Sistema de partículas estilizado via CSS.
* **Cartões de Projetos**: Galeria interativa com efeitos de *hover*.
* **Formulário de Contato**: Envio de mensagens direto para o WhatsApp.

## 🚀 Como rodar o projeto

Para visualizar o projeto localmente em sua máquina, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/LucasPachecoKempfer/portfolio-lucas-pacheco.git](https://github.com/LucasPachecoKempfer/portfolio-lucas-pacheco.git)
    ```
2.  **Navegue até a pasta:**
    ```bash
    cd portfolio-lucas-pacheco
    ```
3.  **Verifique os arquivos:** Certifique-se de que a pasta `img/` contém as imagens citadas na seção de estrutura para que o layout carregue corretamente.
4.  **Execução:** * Basta dar um duplo clique no arquivo `index.html`.
    * Ou utilize a extensão **Live Server** no VS Code para abrir com recarregamento automático.
